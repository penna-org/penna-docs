# 快速开始

## 环境要求

- pnpm >= 8.x (建议使用最新版)
- Node.js  18.x / >= 20.x

## 安装

你可以通过下面的命令来创建一个新的`penna`的项目或者模块。

```shell
# 使用 npm
npm init penna
# pnpm create
pnpm create penna
```
如下：

![penna](/create-penna.png)

## 依赖安装

```shell
# 进入项目目录
cd penna-app

# 安装依赖
pnpm install
```

## 启动项目

```shell
# 启动项目
pnpm dev
```

## 打包项目/模块

```shell
# 构建依赖
pnpm build
```
